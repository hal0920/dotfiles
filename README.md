# dotfiles
 My dotfiles 

## Supported OS 
* MacOS only

## Install

1. Show install list
```sh
cd ~/.dotfiles/
make list
```
2. Make synbolic link  *<font color="DarkGray">(Ex. ~/.vimrc -> ~/.dotfiles/.vimrc)</font>*
```sh
make deploy
```
